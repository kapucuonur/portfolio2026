# Onur Kapucu - Personal Portfolio

This is the personal repository for the portfolio of Onur Kapucu, a Software Engineer and Founder at [TriHonor](https://trihonor.com), building cross-platform solutions for Web, Mobile, and Native Systems (macOS/Garmin).

## Features

- **Modern & Responsive Design:** Built using Tailwind CSS, ensuring a great experience across all devices.
- **Project Showcase:** Highlights featured projects like CoachOnurAI Garmin Bridge, WatchToMac, Ice Bath Tracker, and Learn Finnish.
- **Tech Stack Overview:** Displays skills across Mobile & Native, Frontend, Backend, AI & Cloud, and Creative Tools.
- **Smooth Navigation:** Includes smooth scrolling and mobile-friendly hamburger menus.

## Tech Stack

- **Framework:** [React](https://reactjs.org/) (v18)
- **Tooling:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (v4)
- **Icons:** [Lucide React](https://lucide.dev/)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js and npm (or yarn) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kapucuonur/portfolio2026.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio2026
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`.

## Build for Production

To create a production build, run:
```bash
npm run build
```

To preview the production build locally:
```bash
npm run preview
```
